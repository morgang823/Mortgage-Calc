About The App

The Mortgage Calculator builds a list of payments based on user input and calculates payments out, until the loan is fully paid off.

Skills is used to complete this web application:

  • HTML

    Creates the structure of the web application, create input fields for the user, and a button to submit that information

  • Javascript

    Used to create the function required to take the user's inputs and return a an amortization table of payment information.

  • jQuery

    JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, and CSS animation

  • CSS

    Responsible for the visual desing and appearance of the web application

  • Boostrap

    Built in programming templates allowing for more faster building of tables, forms, and buttons.
